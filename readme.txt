# Бот для напоминаний на Python

## Описание
Этот проект представляет собой Telegram-бота для установки напоминаний. Пользователи могут устанавливать напоминания на заданное количество секунд и получать уведомления об их истечении.

## Функциональность
- Установить напоминание на заданное количество секунд.
- Отменить текущее напоминание.
- Получить уведомление о завершении времени напоминания.

## Установка
1. Убедитесь, что у вас установлен Python 3.7 или выше.
2. Установите необходимые зависимости, выполнив команду:
        pip install python-telegram-bot

## Настройка
1. Замените YOUR_BOT_TOKEN на токен вашего Telegram-бота, который вы можете получить у BotFather.

## Как запустить
1. Откройте терминал и перейдите в директорию, где находится файл.
2. Запустите бота командой:
         python bot.py

## Использование
- Запустите бота и отправьте команду
/start
 для начала работы.
- Выберите действие: установить или отменить напоминание.
- При установке напоминания введите время в секундах, а затем текст напоминания.
- Бот отправит напоминание в указанное время на ваш чат.

## Примечания
- Убедитесь, что бот имеет доступ к интернету и может отправлять сообщения.
- Ограничьте количество активных напоминаний, так как бот использует потоки для их обработки.
